# User registration flow

The user registration flow is designed to allow new users to create an account on the platform. The process involves several steps, including form submission, data verification, and account activation.

## Registration Steps

1. **User Registration Form**: The user fills out a registration form with their email and document number (ID).
2. **Form Submission**: Upon submitting the form, the system validates the input data to ensure it meets the required format and criteria.
3. **Data Verification**: The system checks if the email and document number are unique and not already associated with an existing account.
4. **Account Creation**: If the data is valid and unique, and the user has accepted the terms and conditions, a new user account is created in the database.
5. **Email Confirmation**: An email confirmation is sent to the user's email address to verify ownership. The user must click the confirmation link to activate their account.
6. **Account Activation**: Once the email is confirmed, the user's account is activated, and they can log in to the platform.
